Create `.env` file in server root and provide these credentials for 
```
DATABASE=mongoCredential
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
STRIPE_SECRET=
```

Run for client:
### `npm run start-client`

Run for server:
### `npm run start-server`
